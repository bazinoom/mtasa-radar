# سیستم نقشه - رادار برای ام تی ای MTA SA
![demo][def]
## چگونه بلیپ/آیکون به رادار اضافه کنیم؟
![demo][icon]
برای استفاده از آیکون ها در رادار وارد فایل blips.lua بشید و به این روش بلیپ کریت کنید.
```lua
-- createBlip(x,y,z,icon) -- example
createBlip(1094.2705078125, -1129.98828125, 32.1015625, 13) -- bank
```
با /getpos موقعیت رو کپی کنید و بعد داخل فولدر blips آیکون مورد نظرتون رو پیدا کنید عدد آیکون رو بعد از موقعیت وارد کنید.
